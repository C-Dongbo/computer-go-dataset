SGF
======
[SGF Format](http://www.red-bean.com/sgf/)

1 TYGEM dataset
======
2015.11.02 - 2016.12.31

TYGEM "9D vs 9D" dataset (1,516,031 games).


1.1 Format(index)
------
* id
* date
* white
* white english
* white nation
* black
* black english
* black nation
* komi
* result
* round
* byoyomi/times minutes

nation
* 0 Korea
* 1 Japan
* 2 China
* 3 USA
* 4 Taiwan
* 5 China

1.2 Format(kifu)
------
id\t;B[coord];W[coord];B[coord];W[coord]......
