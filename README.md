TYGEM dataset
======
TYGEM "9D vs 9D" dataset (1,516,031 games).


Format(index)
------
* id
* date
* white
* white english
* white nation
* black
* black english
* black nation
* komi
* result
* round
* byoyomi/times minutes

nation
* 0 Korea
* 1 Japan
* 2 China
* 3 Taiwan
* 4 China

Format(kifu)
------
id\t;B[coord];W[coord];B[coord];W[coord]......
